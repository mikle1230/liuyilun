这两天，我因为vibe coding经常会在 GitHub Trending 上无目的地看，# 当巴菲特被装进一个仓库：关于 ai-berkshire 的一点深夜絮语
凌晨两点，我照例在 GitHub Trending 上无目的地滑着，然后它就那么冒了出来——`xbtlin/ai-berkshire`，一个把"AI 时代的伯克希尔"写进标题的项目。简介只有一行，却让我盯着屏幕愣了几秒：
> 巴菲特·芒格·段永平·李录四大师方法论 + 多 Agent 并行研究。
一个人 + Claude = 一个投研团队。这是作者写在 README 里的slogan。
我没急着读代码，而是先泡了杯茶。因为光是这个名字，就够我想一会儿了。
## 一、四个老人，被写成了四个 prompt
这个仓库做的事，说起来其实很朴素：它把四位价值投资大师——巴菲特、芒格、段永平、李录——各自的方法论，拆解成可调用的 Skill，再派四个 Agent 分别扮演他们，对同一家公司展开独立研究，最后由一个"Team Lead"汇总。四个 Agent 各自搜索、各自判断、互相挑战。
它的卖点是"对抗"而非"分析"。以拼多多为例，段永平的 Agent 说 C2M 模式难复制、给 3.7 分；巴菲特的 Agent 算出扣现金 PE 仅 6.3 倍、给 4.4 分；芒格的 Agent 反过来想护城河其实比想象浅、给 3.5 分；李录的 Agent 盯着管理层文化隐患、只给 2.0 分。巴菲特说"真便宜"，李录说"不确定就不买"——作者认为，这种冲突才是投资决策的真实状态，而单一 prompt 制造不出这种张力。
你大概也有过这样的时刻：读罢巴菲特致股东信，觉得自己懂了；再翻两页芒格的《穷查理宝典》，又觉得刚才那个"懂了"其实只是情绪。四位大师的话，常常是互相打架的。他们真正的智慧，恰恰藏在这种打架里——藏在他们彼此不肯让步的语气、案例的选择、以及那些点到为止的沉默中。
而 ai-berkshire 做的事，是把这种"打架"工程化了。
**我一边觉得这很聪明，一边又有点说不出的怅然。**
## 二、被编码的智慧，与被消解的人
芒格生前反复说过一句话："我这辈子只想知道我会死在哪儿，这样我就永远不去那个地方。"这句话的妙处，不在于它的逻辑，而在于它的语气——一个九十多岁的老人，带着一点冷幽默，把逆向思维讲得像家常。你把它翻译成 prompt 里的"强制思考失败场景，列出 5 大情景及概率"，意思还在，神没了。
段永平在雪球上答网友，经常就三个字："不懂。"或者"10 年。"他几乎从不展开。那种惜字如金背后，是一个人用半辈子交学费换来的笃定。你让一个 Agent 去模拟"段永平视角"，它能复述他的原则，却复述不出那种"我说不懂，就是真的不懂"的分量。
**大师之所以是大师，从来不是因为他们有一套可被复用的方法论，而是因为他们知道在什么时候打破自己的方法论。**
巴菲特一辈子说"不碰科技股"，然后在 2016 年重仓苹果；芒格一辈子讲"反过来想"，晚年却把半生精力押在 Costco 这种"正着想"也一目了然的好生意上。这种自我背叛，是活出来的，不是算出来的。一个 prompt 模板，无论写得多精巧，都很难捕捉到"一个人在七十岁时改变主意"的那种东西。
所以 ai-berkshire 让我想到的，不是一个更聪明的投研工具，而是一个更深的哲学问题：**当智慧被结构化，它还是智慧吗？**
这有点像把一首诗翻译成另一种语言。意思都对了，韵脚也押上了，但读起来就是差一点什么。那点"什么"，是原作者站在某个具体的人生路口、吹着那个路口的风、写下那行字时的全部上下文。它不可移植。
## 三、能力圈的悖论
巴菲特最著名的概念之一，是"能力圈"。他一生的成就，很大程度上来自于他坚决待在自己懂的圈子里，对圈外的东西——哪怕整个世界都在狂欢—— say no。
这里有个特别耐人寻味的悖论。
ai-berkshire 这样的工具，本质上是来帮你"拓宽"能力圈的。你不懂半导体，可以让 Agent 帮你拆解台积电的制程节点；你不懂医药，可以让 Agent 帮你梳理一款创新药的管线进度。**从信息角度，你的圈确实变大了。但从判断角度，它可能反而变小了。**
为什么？因为"能力圈"的核心从来不是信息量，而是"你知道自己不知道什么"。一个真正待在能力圈内的人，他的克制来自于一种切肤之痛——他曾经因为以为自己懂、其实不懂，而亏过真金白银。这种痛感，是圈子的边界。
而 AI 给你的报告，是平滑的、自洽的、带着评分和价格区间的。它不会让你痛。它甚至会让你产生一种"我懂了"的幻觉——毕竟，报告是我"自己"调出来的，结论清清楚楚摆在眼前。
**于是最危险的事情发生了：你以为自己扩大了能力圈，其实你只是把圈的边界，擦得更模糊了。**
这并不是 ai-berkshire 的错。它甚至比大多数 AI 工具都更诚实地面对了这个问题——它内置了"信息丰富度评级"，明确告诉你哪些数据是 B 级、哪些指标是推算的；它强制输出"通过/不通过/灰色地带"，逼你面对非黑即白的决策；它让四个 Agent 互相挑战，就是为了制造那种"我也不确定"的认知摩擦。这些设计，恰恰是在用工程手段，去抵抗 AI 最擅长制造的那种"虚假确定感"。
但工程能抵抗的，终究有限。真正的能力圈，最后还是要靠每个人自己，用时间、用亏损、用错过，一点点刻进骨头里。
## 四、当 AI 版的巴菲特，向阿贝尔提问
写到这里，我忍不住想起 2026 年 5 月那场伯克希尔股东大会。
问答环节的第一个问题，来自一个 AI 合成的"巴菲特"。它说："我是沃伦，来自奥马哈，我 95 岁了，除了时间什么都没有。"然后它问新任 CEO 阿贝尔：为什么股东应该继续长期持有伯克希尔？
阿贝尔后来揭晓，这段视频完全基于公开信息合成，没有用巴菲特本人的任何语音或照片素材。他说，这正是伯克希尔每天面临的现实威胁。
这个场景荒诞得近乎寓言。一个被 AI 复活的巴菲特，坐在自己的追悼会前列，向自己的接班人提问。而台下数万人，那一刻分不清该笑还是该怕。
巴菲特本人对 AI 的态度一直很复杂。他把 AI 比作核武器——"精灵已经从瓶子里出来了，我不知道怎么把它放回去"。但同时，伯克希尔的持仓里，苹果、亚马逊、Snowflake 这些 AI 相关股票早就占了近半壁江山。阿贝尔接任后定的调子是："我们不会为了 AI 而 AI。" 人始终掌握最终决策权，专注狭义 AI，不盲目追求 AGI。
这是一种很"伯克希尔"的态度：既不假装精灵不存在，也不假装自己能驯服它；只是默默地，把它用在铁路调度、保险理赔这些最不起眼的地方。
而 ai-berkshire 这个项目，某种意义上是把同一种态度，搬到了一个普通投资者的书桌上。它不预测股价，不画 K 线，不追热点——它只是认认真真地，用四个老人的口吻，问你一句：这家公司，十年后还在不在？
## 五、我们这一代人的奥马哈
我有时候会想，巴菲特那一代人，他们的"奥马哈"是有地理意义的。一个身处美国中部小城的人，靠读年报、打电话、去现场看工厂，就能跑赢华尔街。那是一种物理意义上的"远离喧嚣"。
而我们这一代人，奥马哈已经不在地图上了。它在终端里，在 prompt 里，在一个又一个被点赞的仓库里。我们以为自己在远离喧嚣，其实我们正坐在喧嚣的最中心——只是喧嚣被包装成了"工具""效率""赋能"。
ai-berkshire 让我着迷的地方，不在于它能帮你赚多少钱（作者晒出 2024 年 +69.29%、2025 年至今 +66.38% 的实盘成绩，但任何理性的读者都知道，历史收益不代表未来）。它真正让我着迷的，是它无意间提出了一个问题：
**当所有大师的方法论都可以被一键调用时，"懂"这件事，还值多少钱？**
我的答案偏悲观。我认为"懂"会越来越不值钱——至少在信息层面。但"信"会越来越值钱。信什么？信你自己亏过的钱、错过的人、熬过的夜。信你自己在某个深夜，看着一份漂亮的 AI 报告，仍然能升起的那种"不对，这里有点问题"的不安。
那种不安，是 AI 给不了你的。也是它永远不该给你的。
README 里引用了巴菲特那句话："Price is what you pay, value is what you get." 这句话放在 AI 时代，似乎多了一层意思：你为一个 AI 工具付出的价格，是 token 费用；而你得到的，是一份看起来无比专业的报告。但那份报告真正的"价值"——它能不能让你在十年后回头时不后悔——取决于你带着什么样的自己，去读它。
工具越锋利，握它的人就越要清醒。
凌晨三点的茶已经凉了。我合上电脑，心里既兴奋又安静。兴奋的是，我们正活在一个一个人就能调起一支投研团队的时代；安静的是，我比任何时候都清楚——
**那个最关键的 Agent，从来不在仓库里，而在屏幕前。**
