最近 AI 编程工具火得一塌糊涂，很多完全不懂技术的朋友跑来问我：“那个传说中能在电脑里自己写代码的 Claude Code，我也能装吗？”
我的回答是：**能，而且没有你想象的那么难。**
你不需要懂编程，不需要懂什么叫“环境变量”，只要你会下载软件、会双击安装、会跟着这篇文章一步步复制粘贴，你就能把这个目前最强大的 AI 命令行工具请到你的电脑里。
下面是专门为“零基础小白”准备的保姆级安装教程，适用于 Windows、Mac 和 Linux。
---
## 准备工作：你需要一个 Anthropic 账号
Claude Code 是 Anthropic 公司出品的工具。在安装它之前，你需要先有一个他们的账号。
1. 打开浏览器，访问 [Anthropic 官方控制台](https://console.anthropic.com/)。
2. 注册一个账号（就像注册任何一个普通网站一样）。
3. 登录进去。不用担心里面复杂的设置，只要账号能登录就行。稍后 Claude Code 会自动跳转到这个网页来完成授权。
准备好账号后，我们正式开始。
---
## 第一步：安装“引擎”——Node.js
Claude Code 是用一种叫做 JavaScript 的语言写的。要让它在你的电脑上跑起来，你需要先安装一个“翻译官”，官方名称叫 **Node.js**。
这一步非常简单，就像平时下载安装微信一样。
### Windows 系统
1. 打开 [Node.js 官网](https://nodejs.org/)。
2. 你会看到两个绿色的大按钮，选择左边那个标有 **LTS（长期支持版）** 的按钮点击下载。
3. 下载完成后，双击运行那个 `.msi` 安装包。
4. 接下来就是无脑点击 **Next（下一步）**，一路点 **I Agree**、**Next**，最后点击 **Install**。如果弹出权限提示，选择“是”。
5. 点击 **Finish** 完成安装。
### Mac 系统
1. 同样打开 [Node.js 官网](https://nodejs.org/)。
2. 点击左边 **LTS** 按钮下载安装包。
3. 下载完成后，双击 `.pkg` 文件。
4. 弹出安装向导后，一路点击 **继续**，然后 **同意**，最后 **安装**。系统可能会要求你输入电脑的开机密码，输入后等待安装完成即可。
### Linux 系统
如果你在使用 Linux，你大概率已经有一定的折腾能力了。最简单的方式是通过包管理器安装：
以 Ubuntu/Debian 为例，打开终端输入：
```bash
sudo apt update
sudo apt install nodejs npm
```
（如果你是其他发行版，请使用对应的包管理器安装 nodejs 和 npm）。
---
## 第二步：打开那个“黑色框框”——终端
Claude Code 是一个命令行工具，这意味着它没有花里胡哨的界面，你需要在一个叫“终端”的黑色框框里和它对话。别怕，它只是个输入文字的地方。
### Windows 系统
1. 按下键盘上的 `Win` 键（带有 Windows 图标的键）。
2. 直接输入 `PowerShell`。
3. 在搜索结果中点击 **Windows PowerShell**。如果你看到蓝底白字的窗口弹出来，就成功了。
### Mac 系统
1. 按下键盘上的 `Command + 空格键`，调出聚焦搜索。
2. 输入 `终端` 或者 `Terminal`。
3. 按下回车键，弹出一个白色（或黑色）的小窗口。
### Linux 系统
按下 `Ctrl + Alt + T`，终端就会弹出来。
---
## 第三步：念动咒语——安装 Claude Code
现在，你的终端已经打开了。我们将通过一条简单的命令来安装 Claude Code。
在终端里，**完整复制并粘贴**下面这行代码：
```bash
npm install -g @anthropic-ai/claude-code
```
> **小提示**：
> - `npm` 意思是包管理器，`install` 是安装，`-g` 代表全局安装（也就是你的整台电脑都能用）。不用死记硬背，复制粘贴就行。
> - 在 Windows 的 PowerShell 里，粘贴可以用鼠标右键单击。
> - 在 Mac 和 Linux 里，粘贴可以用 `Command + V` 或 `Ctrl + Shift + V`。
粘贴好后，按下**回车键**。
此时，屏幕上可能会出现一堆滚动的代码，看起来很吓人，但实际上它正在努力下载和安装。**你只需要耐心等待一两分钟，不要关闭窗口。** 当屏幕不再滚动，重新出现可以输入文字的提示符时，说明安装完成！
---
## 第四步：唤醒 Claude Code 并登录
安装已经完成了！现在，我们来叫醒它。
在终端里输入这四个字母：
```bash
claude
```
按下**回车键**。
第一次运行时，Claude Code 会要求你登录验证。你的终端里会出现一段提示，大概意思是让你按回车键去网页授权。
1. 按下回车键，你的浏览器会自动打开一个网页。
2. 这个网页就是刚才你注册的 Anthropic 控制台页面。
3. 网页上会问你是否允许 Claude Code 访问，点击 **Authorize（授权）** 或类似的同意按钮。
4. 网页上可能会显示一串代码，让你复制回终端；或者直接提示“授权成功”。
5. 回到你的终端窗口。
如果一切顺利，你会看到终端里出现了一个欢迎界面，或者是一个等待你输入的提示符。比如它可能会说：“How can I help you today?”
---
## 恭喜你，完成了！
虽然你一行代码都没写过，但你刚刚完成了一次开发者级别的环境配置！
现在，你可以试着在终端里用自然语言和它聊天了。比如你可以输入：
> 帮我在桌面上创建一个文件夹，名字叫“我的AI作品”
按下回车，你就会看到 Claude Code 开始自己在终端里敲打命令，帮你把事情办妥。
### 如果以后不想用了怎么退出？
在终端里输入 `exit` 然后按回车，或者直接按 `Ctrl + C` 两次，就可以关掉 Claude Code 了。
